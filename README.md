# AI-Based Phishing Email Detection System

This project is a machine learning-based system designed to detect phishing emails.

## Features
- Email classification (Phishing or Safe)
- TF-IDF feature extraction
- Logistic Regression & Random Forest models
- Confidence score output
- Flask web application interface

## Technologies Used
- Python
- Scikit-learn
- Flask
- HTML/CSS

## How It Works
Users input email text into the web application, and the system predicts whether it is phishing or legitimate, along with a confidence score.
